# 🏦 BankPulse AI

**📈 Predict the Rush. 🛡️ Prevent the Crash.**

AI-powered Banking Infrastructure Observability & Downtime Prevention Platform

---

## ❓ Problem

Banking systems fail during traffic spikes — salary credits, GST deadlines, festival shopping, or cyberattacks. Most tools react after infrastructure breaks, leading to ❌ failed transactions, ⏱️ high latency, and 💸 revenue loss.

## 💡 Solution

BankPulse AI simulates real-time banking infrastructure, detects anomalies, predicts crash probability, and generates automated mitigation plans — ⚡ before failure happens.

## 🔧 Key Capabilities

| Feature | Description |
|---------|-------------|
| 📡 Live Monitoring | Transaction throughput, latency, CPU/RAM, DB saturation, branch traffic |
| 🧠 AI Crash Prediction | Risk scoring, threat level, recovery ETA, mitigation plans |
| 🛡️ DDoS Detection | Layer-7 flood detection, traffic entropy analysis |
| 🔄 Failover Simulation | Primary → Secondary recovery with zero data loss |
| 🌍 Branch Network | Track 8+ metro branches in real time |
| 🤖 GenAI Copilot | Auto-generates incident plans and customer alerts |

## 🎬 Simulated Scenarios

📅 Normal Day · 📆 Mid-Month · 🧾 GST Filing · 💰 Salary Day · 🎉 Festival Season · 🔥 Flash Sale · 📉 Year End · 💀 DDoS Attack

## ⚙️ Tech Stack

- 🎨 **Frontend:** Streamlit, Custom CSS
- 🐍 **Backend:** Python, State Sync Engine
- 🧠 **AI:** Google Gemini API (offline fallback)
- 🗃️ **Data:** Centralized session state

## 🧪 Demo Flow

1️⃣ Connect to simulated core banking system  
2️⃣ Observe normal operations  
3️⃣ Trigger peak scenario (Salary/GST/Festival)  
4️⃣ Watch load, latency, and alerts rise

6️⃣ Trigger failover → system recovers in seconds
